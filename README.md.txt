ShadowJero Battle

¡Bienvenidos al juego de batalla entre Kirby Diablo Supremo y ShadowHero.exe!

Este juego permite realizar ataques y ver cómo evoluciona la batalla en tiempo real.

Cómo jugar:

Haz clic en los botones de ataque para reducir la vida del oponente.

¡El primero en llegar a 0 HP pierde!

Tecnologías usadas:

HTML

CSS

JavaScript
